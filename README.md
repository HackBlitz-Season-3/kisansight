# 🌾 KisanSight — Sell/Hold Decision Assistant for Indian Farmers

---

## 📁 Project Structure

```
kisansight/
├── backend/
│   ├── main.py              # FastAPI app — all routes
│   ├── prophet_model.py     # Prophet forecasting engine
│   ├── weather.py           # Open-Meteo API (free, no key)
│   ├── news.py              # NewsData.io (free tier)
│   ├── explain.py           # Gemini API explanation layer
│   ├── prepare_data.py      # One-time: XLSX → CSV converter
│   ├── requirements.txt
│   └── data/
│       └── prices.csv       # Generated by prepare_data.py
│
└── frontend/
    ├── src/
    │   ├── App.jsx           # Root + language toggle
    │   ├── i18n.js           # Hindi / Marathi / English strings
    │   ├── pages/
    │   │   └── Dashboard.jsx # Main page
    │   └── components/
    │       ├── SignalBadge.jsx   # Big SELL/WAIT signal
    │       ├── ForecastChart.jsx # Chart.js price graph
    │       ├── WeatherCard.jsx   # 5-day weather
    │       └── NewsCard.jsx      # Latest market news
    ├── package.json
    └── vite.config.js        # PWA + Tailwind setup




```bash
cd backend

# Create virtual environment
python -m venv venv
source venv/bin/activate      # Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Prepare dataset (run once)
python prepare_data.py path/to/agriprice_dataset.xlsx

# Create .env file
echo "GEMINI_API_KEY=your_key_here" > .env
echo "NEWSDATA_API_KEY=your_key_here" >> .env

# Start server
uvicorn main:app --reload --port 8000
```

Backend runs at: http://localhost:8000
API docs at: http://localhost:8000/docs



```bash
cd frontend

npm install
npm run dev
```

Frontend runs at: http://localhost:5173
